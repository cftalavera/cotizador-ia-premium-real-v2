
# COTIZADOR IA PREMIUM + INSTALADOR

- instalar_windows.bat

---

# WINDOWS

1. Extraer:
cotizador-ia-premium-real.zip

2. Ejecutar:
instalar_windows.bat

3. Crear:
backend/.env

4. Agregar:
OPENAI_API_KEY=sk-tu-api-key

5. Ejecutar:
ejecutar_windows.bat

# LINUX / MAC

Dar permisos:

chmod +x instalar_linux.sh
chmod +x ejecutar_linux.sh

Instalar:

./instalar_linux.sh

Ejecutar:

./ejecutar_linux.sh
